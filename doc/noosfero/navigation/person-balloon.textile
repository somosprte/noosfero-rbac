h1(order-618). Person balloon

p.  _When an user is navigating on the system, he can see details of a person and become a friend quickly._

*  "Description":#description
*  "Related topics":#relatedtopics

h2(#howaccess). How to access

p. If enabled on environment, wherever a person appears you can click on the top-left corner to see the person balloon !=/images/doc/person-balloon-closed-en.png(Person balloon link)!

h2(#description). Description

p. The person balloon has links to allow direct access to some person's pages !=/images/doc/person-balloon-en.png(person balloon links)!
# Wall: lists scraps sent to this person
# Friends: lists the person's friends
# Communities: lists the person's communities
# Add: if you are not friend of the person, you can quickly ask to add her clicking on this link
# Send an e-mail: if you are a friend of the person, you can send an e-mail to the her clicking on this link
